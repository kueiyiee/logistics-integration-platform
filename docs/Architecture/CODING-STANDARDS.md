# Coding Standards

## Backend

- Follow PSR-12 and Laravel conventions.
- Keep controllers thin and push orchestration into services.
- Use form request validation and policy checks for mutations.
- Prefer explicit naming for repositories, jobs, events, and listeners.

## Frontend

- Use TypeScript strict mode.
- Keep components focused and composable.
- Separate server-state fetching from presentational components.
- Standardize UI patterns for tables, forms, and empty states.
