# Delivery Portal

Enterprise SaaS platform for third-party delivery integration, designed for a multi-tenant, API-first, production deployment model.

## Architecture at a glance

- Backend: Laravel 13, PHP 8.5, MySQL 8, Redis, queues, Sanctum, event-driven services.
- Frontend: React 19, TypeScript, Vite, Tailwind CSS v4, shadcn/ui, Radix UI, Framer Motion.
- Deployment: Docker, Docker Compose, Nginx, Apache, GitHub Actions, environment separation.
- Docs: Architecture, API, database, deployment, security, and team workflow.
